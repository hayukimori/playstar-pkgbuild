# playstar-pkgbuild

Arch Linux PKGBUILD for [PlayStar](https://github.com/hayukimori/PlayStar), a lightweight local music player built with Godot and LibVLC.

## Installation

```bash
git clone https://github.com/hayukimori/playstar-pkgbuild
cd playstar-pkgbuild
makepkg -si
```
